# meow-names

> Get popular names for your pet cat

![](myMolly.jpeg)


## Install

```
$ npm install --save meow-names
```


## Usage

```js
const meowNames = require('meow-names');

meowNames.random();
```


## API

### .all

Type: `array`

All cat names in alphabetical order.

### .random()

Type: `function`

Random cat name.
